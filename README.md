# LitAPI_JAVA
 用Java描述洛阳理工学院。由于项目的特殊性，随缘维护。服务效率取决于校服务器。  



## 项目环境  
- JDK: 开发版本为OpenJDK11（因为本人是VScode的忠实拥趸，而这个版本的JDK正是VScode中Java插件推荐的版本）
- 项目管理工具: Maven1.4
- 依赖包:  
    - fastjson-1.2.79
    - jsoup-1.14.3
